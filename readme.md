## Explanation of what the game is and how to play (2-3sentences, please)
    - 1. First of all, our game uses the a, d and w arrow keys in the bottom view and the w, a, s, d arrow keys and shift in the top view.

    -2. Side view (W): Jump

    -3. Top view (shift + w,a,s,d): dash

    -4. all view (space bar): chang view

    -Tip If you jump, quickly switch screens, and then dash, you can move further or go to a place slightly above you!

    -Tip Even if you jump and switch screens, it doesn’t work!

    -Tip You can freely move the map using the up and down buttons.

    -Tip If you use a dash, it can skip the short wall
